## 当前问题

### OpenAI API 代理服务器不稳定

**现象**：调用 AI 生成题目时，频繁出现 504 Gateway Time-out 错误。

**原因**：使用的 OpenAI API 代理地址 `https://ai.gxwtf.cn/v1` 服务器不稳定，导致请求超时。

**解决方案建议**：
1. 检查代理服务器状态或更换更稳定的代理服务
2. 确保 `.env.local` 中的 `OPENAI_API_KEY` 和 `OPENAI_API_BASE` 配置正确
3. 增加超时时间或重试次数（已经在代码中添加）
