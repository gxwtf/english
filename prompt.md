你需要实现下列功能（写入 ./src/page.tsx 文件中）：

你需要判断用户是否登录（判断登录你可以暂时预留，不实现）

如果已登录：

- 创建一个导航栏，包含两项：“我的单词本” 和 “题目列表”。其中“我的单词本”指向当前页面，而“题目列表”指向一个链接 https://example.com。
- 页面的主体是一个单词列表，显示用户已添加的单词。每个单词包括：
    - 英文单词；
    - 中文释义（小字，写在英文单词下方）；
    - 标签列表（写在单词右侧）；
    - 操作列表。操作包括：选中单词（最左侧）；编辑单词；删除单词（弹窗确认）。
- 单词列表上方可以进行操作：全选；按单词首字母排序；默认排序；按标签筛选（支持“与”和“或”）；AI 出题（一个按钮，需要选中至少一个单词才能选择该按钮）；删除选中的单词（弹窗确认）。
- 有一个“添加单词”的按钮，点击后出现一个弹窗，用户需要输入要添加的单词，系统需要调用 `./src/api/dict/query.ts` 下的查词 API 显示该单词的所有中文释义和词性，用户需要从中选择自己不熟悉的释义（可以全选），至少选择一个。然后给该单词选择标签，最后添加单词。
- 编辑单词弹窗与添加单词类似。

如果未登录：

在页面中央显示大标题“广学英语”，副标题“智能、高效的英语单词记忆软件”。

下方显示该产品的优势：

1. 支持一词多义；
2. 支持 AI 出题，对标高考题型。

页面尽可能美观，操作尽可能人性化，多添加一些图标。

使用 React + Next.js + Typescript TSX + shadcn-ui + lucide-react。

编写完成之后，你需要进行测试，可以使用 Chrome MCP。你需要全面测试各个功能，测试越全面，考虑的 Corner case 越多，加分越多！
